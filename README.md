# WebReadProcess 隐私政策

生效日期：2026 年 6 月 30 日

WebReadProcess 是一款 Safari 阅读进度扩展，用于在用户指定的网站上记录阅读位置，并帮助用户快速回到上次离开的位置。

我们重视用户隐私。本隐私政策说明 WebReadProcess 如何处理数据。

## 1. 我们不收集个人数据

WebReadProcess 不会将用户数据上传到我们的服务器，也不会将用户数据发送给第三方。

本扩展不会收集、出售、共享或追踪以下信息：

- 姓名、邮箱、电话号码等联系信息
- 精确位置或粗略位置
- 设备标识符或广告标识符
- 浏览历史或搜索历史
- 崩溃日志、分析数据或诊断数据
- 用于广告、营销或用户画像的数据

WebReadProcess 不包含广告 SDK、分析 SDK 或第三方追踪 SDK。

## 2. 本地保存的阅读记录

为实现阅读进度恢复功能，WebReadProcess 会在用户设备本地保存以下信息：

- 用户在扩展设置中添加的跟踪网站规则
- 已记录页面的 URL
- 页面标题
- 页面域名
- 页面滚动位置
- 阅读进度百分比
- 最近更新时间

这些信息仅保存在 Safari 扩展的本地存储中，用于在用户重新打开同一页面时恢复阅读位置，并在扩展弹出窗口中显示最近阅读记录。

这些数据不会离开用户设备，不会被上传、同步、出售或共享给任何第三方。

## 3. 网站访问权限

WebReadProcess 需要访问用户指定的网站页面，才能读取当前页面地址、标题和滚动位置，并在页面再次打开时恢复滚动位置。

扩展只会对用户在设置页中启用的站点执行阅读进度记录。对于未启用的网站，扩展不会保存阅读进度。

## 4. 第三方服务

WebReadProcess 不使用第三方分析服务、广告服务、崩溃报告服务或远程数据库。

用户访问的网站本身可能有独立的隐私政策和数据处理行为。WebReadProcess 无法控制这些网站的行为，也不代表这些网站处理数据。

## 5. 数据删除

用户可以在扩展中删除单条阅读记录，或在设置页中清空全部阅读记录。

用户也可以通过 Safari 的扩展管理、网站数据管理或卸载应用来移除 WebReadProcess 保存的本地数据。

## 6. 儿童隐私

WebReadProcess 不面向儿童收集数据，也不会有意收集儿童的个人信息。

## 7. 政策变更

如果 WebReadProcess 的数据处理方式发生变化，我们会更新本隐私政策，并在适当位置更新生效日期。

## 8. 联系方式

如果你对本隐私政策或 WebReadProcess 的隐私实践有任何问题，可以通过以下邮箱联系我们：

# WebReadProcess Privacy Policy

Effective Date: June 30, 2026

WebReadProcess is a Safari reading progress extension that records reading positions on websites selected by the user and helps users return to where they left off.

We respect user privacy. This Privacy Policy explains how WebReadProcess handles data.

## 1. We Do Not Collect Personal Data

WebReadProcess does not upload user data to our servers and does not send user data to third parties.

The extension does not collect, sell, share, or track:

- Contact information such as name, email address, or phone number
- Precise or coarse location
- Device identifiers or advertising identifiers
- Browsing history or search history
- Crash logs, analytics, or diagnostic data
- Data used for advertising, marketing, or profiling

WebReadProcess does not include advertising SDKs, analytics SDKs, or third-party tracking SDKs.

## 2. Reading Records Stored Locally

To provide reading progress restoration, WebReadProcess stores the following information locally on the user's device:

- Website rules configured by the user
- Recorded page URL
- Page title
- Page domain
- Page scroll position
- Reading progress percentage
- Last updated time

This information is stored only in Safari extension local storage. It is used to restore the reading position when the user reopens the same page and to show recent reading records in the extension popup.

This data does not leave the user's device and is not uploaded, synced, sold, or shared with any third party.

## 3. Website Access

WebReadProcess needs access to selected website pages to read the current page URL, title, and scroll position, and to restore the scroll position when the page is opened again.

The extension records reading progress only for sites enabled by the user in the options page. For sites that are not enabled, the extension does not save reading progress.

## 4. Third-Party Services

WebReadProcess does not use third-party analytics services, advertising services, crash reporting services, or remote databases.

Websites visited by the user may have their own privacy policies and data practices. WebReadProcess does not control or represent those websites.

## 5. Data Deletion

Users can delete individual reading records in the extension or clear all reading records from the options page.

Users can also remove locally stored WebReadProcess data by managing Safari extensions, managing website data, or uninstalling the app.

## 6. Children's Privacy

WebReadProcess does not collect data from children and does not knowingly collect personal information from children.

## 7. Changes to This Policy

If WebReadProcess changes its data handling practices, we will update this Privacy Policy and revise the effective date where appropriate.
